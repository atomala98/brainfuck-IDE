# Brainfuck IDE

Program created for writing code in brainfuck programming language. Program enables writing code, adding inputs, storing code in txt files. You can download standalone version of interpreter right here: https://www.dropbox.com/s/mckt277rtkfjdm5/brainfuck%20IDE.rar?dl=0 (run app.exe).

All planned inputs should be added in ASCII (0 - 256), one under the other in separate lines.
